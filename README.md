# Challenge 2 : Stock Analysis Using Excel VBA

**This project contains analysis of stock market generated data using Excel Visual Basic for Applications (VBA)**

This Repo contains four files: 
-  StockMarketAnalysis.vbs : This file contains code for stock market analysis designed in VBA.
-  Year2018_solution.png : Screenshot of stock market analysis for year 2018.
-  Year2019_solution.png : Screenshot of stock market analysis for year 2019.
-  Year2020_solution.png : Screenshot of stock market analysis for year 2020.

References used  :
